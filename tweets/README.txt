Twitter Tweets extract
Τρέχουμε: python tweets.py αφού έχουμε εγκαταστήσει το tweepy.
Το script παίρνει από το χρήστη σαν είσοδο δύο χρήστες στο twitter και βρίσκει ποιος από τους δύο χρησιμοποίησε τις 
περισσότερες λέξεις στα 50 τελευταία του tweets.
Πατάμε y ή n για να τυπώσουμε κάθε λέξη που μετρήθηκε.
Στη συνέχεια γράφουμε δυο ονόματα twitter όπως εμφανίζονται σαν screen names και πατάμε Enter:
Δηλαδή ενώ στο twitter βλέπω @NASA ή @elonmusk, θα πληκτρολογήσω:
NASA και Enter
elonmusk και Enter
